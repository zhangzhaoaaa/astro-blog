> 摘要：腾讯门口排队装龙虾，闲鱼代装从500跌到9.9，这其中最荒诞的冲突在于：当近千个普通人在排队抢夺这场“改变一切”的红利时，那个烧掉400美元死磕自动化的极客打出了6.5的失望分，而创造它的天才已经转身离去。OpenClaw的狂热，值得分析的不是它的技术，而是我们的慌张。

## 腾讯门口的一千人和离开的创造者

2026年3月6日，深圳腾讯总部门口排起了近千人的长队。这不是发烧友彻夜排队买新iPhone，而是等候腾讯云的工程师帮他们免费在电脑里装上一只“小龙虾”（OpenClaw）。队伍里混杂着格子衫的程序员、穿着校服的小学生，还有拿着手机拍照的大爷大妈

对于他们中的许多人来说，这只虾代表着“不掉队”

但这幅充满市井狂热的画面，和另外两个切面形成了巨大的张力：

在大洋彼岸，Peter Steinberger——那个用短短几个月斩获数十万Star，把OpenClaw推上神坛的奥地利老哥，在整个中文互联网圈子正为他疯狂时，做出了一个反直觉的决定：他把项目的所有权捐给了独立基金会，自己跑去了 OpenAI 打工。他在博客里留下一句意味深长的话："我完全能看到它怎么变成一家大公司。但不，这对我来说并不兴奋。"

在他离开的同时，一位技术博主 ssntpl.com 的测试者刚刚结束了长达数天的极限评估。为了验证OpenClaw是否真的能像传闻中那样“一键全自动”，这位极客硬生生烧掉了 400 美元的 API Token。最后的结论是什么？

他在长文结尾重重地敲下两行字：综合评分 **6.5/10**
**"它能减少工作量，但不能免除责任。"**

一头是沸腾的全民朝圣，一头是创造者的离去与极客的冷水
同一个工具，为什么会发生劈裂度如此之高的社会反差？

## 焦虑变现：从 500 元跌到 9.9 元的罗生门

答案不在代码结构里，而在“养虾”产业链的账本里

OpenClaw 是一款在本地运行、接入大模型并能够自主在电脑上执行操作的开源工具。它和 ChatGPT 的区别在于：ChatGPT 是“你问它答”，而 OpenClaw 是“你说一句，它替你把整件事办完”

这不仅是技术的范式转移，更是焦虑的放大器。当”AI 替你把整件事办完”的短视频切片在算法助推下铺天盖地时，无数深夜刷到视频的职场人感到了前所未有的生存恐慌

恐慌立即变成了生意。闲鱼和小红书成为了这场狂热的前沿阵地。仅仅两周时间，闲鱼上“代装 OpenClaw”的报价经历了一次自由落体：从最高的 500 元，断崖式跌到了 9.9 元

跌至9.9元不是因为技术门槛消失了，而是因为最恐慌的那批急于上车的人已经被收割完毕，产业链迅速下沉分化。卖铲子的人不再只靠代装赚钱，他们演变出了：

1. **代装入口**（9.9元赚个吆喝以筛选焦虑客）
2. **付费社群**（99-199元的“养虾变现互助群”，进去才发现没人说话）
3. **定制服务**（拿着免费的开源 Skill，反手标价三五千块卖给小白）

那个在抖音上声称自己靠上门装机赚了26万的大V，才是这场游戏里最确定赚到钱的角色之一。至于那些排队上车、花钱买心安的普通人，大多数人在装完程序的第二个周末，看着空转的 API 和难以调教的报错面板，发出了灵魂拷问：“装好了，然后呢？”

## 狂热的引擎：云厂商与AI叙事的共谋

如果只是一群焦虑的普通人在折腾，事情不会闹得这么大。腾讯那门口的一千人排队，揭秘了幕后的超级推手——云厂商的补贴战

阿里云、腾讯云、字节火山引擎，中国云巨头史无前例地在几天内同步上线了“OpenClaw 一键部署方案”。腾讯甚至搞出了现场装机的奇观

这是一场阳谋。因为运行 OpenClaw 需要服务器、需要长期运行的计算资源。对于云平台来说，这是完美的算力获客钩子。不管这只龙虾在你手里到底能不能接单赚钱，只要你把它养在了云服务器上，就意味着你每月将源源不断地支付“饲料费”（云资源费）

借着 2026 年初 DeepSeek 吹起的“AI 平权”大风，OpenClaw 这个免费的、个人的开源 Agent，成为了云厂商兜售算力的最佳叙事容器。连深圳的区级政务部门，都迅速上线了基于类似架构的"政务龙虾"试点

每个人都拿到了自己想要的：大V赚了焦虑的钱，云厂商拿到了月活用户，吃瓜群众获得了”我离未来不远了”的心理幻觉。只有那个烧了 400 美元的测试极客，在承受真实代价

## 同一只虾，两个世界

但这里有一个几乎所有中文讨论都跳过的问题：OpenClaw 是全球开源项目，为什么”全民养虾”只发生在中国？

OpenClaw 在美国同样很火——它是 GitHub 上增长最快的开源项目之一，Hacker News 连续多周讨论不断，Cisco、卡巴斯基等安全巨头专门发布了风险评估报告。但美国的”火”，始终停留在开发者社区内部。最高赞的评论冷冷地写道：”这些工具做重复性任务还行，碰到真正困难的开发就崩了。”讨论完，大家散场继续写代码。没有排队，没有代装，没有股票涨停

同一个工具，在美国是开发者论坛里的技术辩论，在中国是腾讯门口近千人的长队、闲鱼上几万单的代装生意、和云厂商集体下场的补贴大战。差别不在热度，在于它从”开发者工具”变成了”全民运动”

这不是因为中国人更”不理性”。这是因为同一颗火星，落进了完全不同的火药桶里

**这口锅里至少有三味独特的料：**

**第一味，是高度紧绷的就业竞争。** 2026年，1222万应届毕业生涌入市场，史上最多。传统吸纳高学历人才的行业正在同时调整。在这种环境下，”学会AI”从个人兴趣变成了职业焦虑的出口——不是因为真的想用，而是怕不会用就掉队

**第二味，是短视频的情绪核弹。** 抖音和快手的算法分发能力，和 Hacker News 的文字讨论区完全不在一个量级。一段十秒钟的”AI自动排版财报”视频，24小时内可以触达数千万人。而算法奖励的是情绪峰值，不是准确性——所以你只会刷到那次完美成功的录屏，永远刷不到它背后十几次卡死报错的废弃录像

**第三味，是闲鱼式的零工经济。** 中国有一个西方完全不存在的基础设施：成熟的零工服务市集。焦虑可以在48小时内被打包成一门9.9元的生意。OpenClaw 自己的维护者曾在 Discord 上说过一句大实话：”如果你连命令行都不会用，这个项目对你来说太危险了。”但中国市场直接跳过了这个前提——一个需要开发者能力的工具，被卖给了连终端都没打开过的人

还有第四味，我不敢写，懂的自然懂

光是这三味凑齐，已经足够引爆。更别提还有那不能写的第四味

所以，”养虾”狂热的真正主语不是 OpenClaw，而是这套独特的焦虑放大机制。这只虾只是恰好在这个时间点，落进了这口锅里

## 能力的楚河汉界：能做什么，不能做什么

去掉狂热的外衣，我们必须冷静正视用 400 美元账单换来的教训：边界在哪？

**能做的（真实好用的效率区间）：**

- **邮件整理和碎片回复**：当任务结构化、容错率高（草稿错一封也不致命）时，它是神器
- **文件格式转换与知识聚类**：处理机械性文档，省去枯燥点击
- **代码辅助与原型搭建**：让它帮你写一段脚本、搭一个网页原型，速度确实比从零开始快得多
- **信息搜集与初步整理**：把散落在各处的资料拉到一起形成初稿，再由人来判断和修正

这些场景的共同特征是：错了可以改，改了代价不大。在这个区间里，OpenClaw 确实是一个值得尝试的效率工具

**做不到的（结构性限制，当员工用是赌博）：**

- **可靠的连贯操作**：实测中，让它自动化注册 GitHub 账号失败率极高（单次成本4.5美元），自动化代发社交媒体更曾花了 18 美元
- **跨天上下文的记忆连续性**：如果一件事超过两天，它的记忆会静默衰退。
- **安全的执行环境**：Cisco 审计发现 512 个隐患；上海科技大学给出的安全通过率仅 58.9%；微软直言这是“具有持久凭证的不受信任代码”

**这里有一条清晰的判断标尺：**
如果是“做错了大不了撤销重来”、“不涉及客户商业机密”的事，OpenClaw 是好用的私人助理。如果是“每次结果必须 100% 确定”、“执行出错面临严重亏损”的事，那将它交给那只小龙虾，不叫自动化，叫扔硬币赌博

## 尾声

在 LINUX DO 论坛里，有一条被疯狂点赞的帖子，替所有冷静的旁观者说出了心里话：
**”满屏的教程都在教你怎么按步骤一二三装上它，但你永远搜不到：装完之后，他们到底拿它做了什么有价值的事？”**

热搜上，公务员”养龙虾”的热潮已经在滑落。闲鱼上的代装虽然只需 9.9 元，但付款的数字已经几天没有跳动了

回望文首，创造者 Peter Steinberger 选择去 OpenAI 制造成熟体系的产品，而不是固守这只不可控的”野生龙虾”。他大概比任何人都清楚：从一个伟大的开源框架，到一个能真正替代人类执行确定性业务的产品之间，差的根本不是几十万个 Star，而是漫长、枯燥甚至满是报错的工程跋涉

对于真正有痛点的技术向从业者而言，OpenClaw 会渐渐变成他们系统中沉默但高效的一行脚本。但对于更多被 FOMO 裹挟进来的普通人而言，那只曾在屏幕上疯狂移动鼠标的小龙虾，最终极有可能像曾经囤积的电子书和跑步机一样，安静积灰

而这场实验最终会沉淀出什么，不取决于腾讯总部门口排了多长的队，而取决于那些看着报错停机的职场人，今晚会选择关掉它叹气睡觉，还是选择打开代码去 debug 的一闪念间
